# Personalized Workout & Diet Planner with AI

## Overview

Personalized Workout & Diet Planner with AI is a web-based application that generates customized workout routines and diet plans based on a user's fitness goals, budget, dietary preferences, available equipment, and health conditions.

## Problem Statement

Most fitness applications provide generic workout and diet recommendations that do not consider individual needs, cultural food habits, or available resources. This project aims to provide personalized and practical fitness guidance using Artificial Intelligence.

## Features

* Personalized workout recommendations
* Personalized diet plans
* BMI and calorie calculations
* User authentication
* Fitness profile management
* AI-based recommendation engine

## Technology Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Python, Flask
* Database: SQLite
* AI/ML: Scikit-learn, Pandas, NumPy

## Installation

```bash

cd Personalized-Workout-Diet-Planner-with-AI-
pip install -r requirements.txt
```

## Run the Project

```bash
python backend/app.py
```

Open the frontend in your browser and start using the application.

## AI Module

The AI engine analyzes user information such as:

* Age
* Gender
* Height
* Weight
* Fitness Goal
* Activity Level
* Budget
* Dietary Preference
* Available Equipment
* Health Conditions

Based on these factors, it recommends suitable workout and diet plans.
