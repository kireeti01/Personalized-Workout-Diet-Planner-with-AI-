# Dataset Design

## User Profile Structure
The dataset captures the following fields:
- Age
- Gender
- Height
- Weight
- BMI
- Fitness Goal
- Activity Level
- Budget
- Dietary Preference
- Available Equipment
- Health Conditions
- WorkoutCategory
- MealCategory

## Purpose
The `training_data.csv` file contains synthetic data used to train the AI recommendation models. The fields are balanced across typical student fitness goals.
