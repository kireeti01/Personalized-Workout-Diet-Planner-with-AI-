# Project Documentation

## Abstract
The Personalized Workout & Diet Planner with AI is a student-centric fitness system that moves beyond generic plans. It uses structured profile data and AI-driven reasoning to recommend safe, affordable, and culturally aligned workout and diet plans.

## Introduction
Most fitness solutions ignore individual difference in budget, culture, equipment access, and medical restrictions. This system solves that gap by combining user profile inputs with intelligent recommendations.

## Literature Survey
Existing fitness apps rely largely on predefined templates. Research shows personalization improves adherence and results. This project bridges user preferences with an automated recommendation engine.

## Methodology
1. Capture user personal and health details.
2. Calculate BMI and daily calorie needs.
3. Use rule-based mapping and ML classification for recommendation categories.
4. Save user profiles to SQLite and provide a dashboard view.

## Results
The system produces:
- Personalized workout routines
- Tailored meal plans
- Goal-specific advice for loss, maintenance, and gain

## Conclusion
The project demonstrates a practical AI-assisted wellness planner suitable for academic submission and real-world prototypes.
