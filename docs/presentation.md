## Viva Questions
1. What is the main goal of this project?
   - To create personalized workout and meal plans based on individual student profiles.
2. Why did you choose SQLite for the database?
   - SQLite is lightweight, easy to configure, and ideal for a prototype or academic project.
3. How does the AI module work?
   - It calculates BMI, BMR, daily calories, and uses a trained classification model to suggest workout and meal categories.
4. What are the key inputs for recommendation?
   - Age, gender, height, weight, goal, activity level, budget, dietary preference, equipment, and health conditions.
5. How would you deploy this project?
   - Backend on Render or Railway, frontend as static files or GitHub Pages, with updated API endpoint configuration.
